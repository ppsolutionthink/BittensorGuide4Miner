# BittensorGuide4Miner
Bittensor Guide For Miner

by @ppsolutionthink


**Bittensor Blockchain Explorer** https://taostats.io/
### Prerequirements
- python3.10
- python3.10-venv
- python3-pip

``` 
(optional)
$ alias python=python3
```

``` 
(optional)
$ python -m venv venv
$ source venv/bin/activate
```

### Install `btcli`
``` 
$ pip install bittensor-cli
```

#### check `btcli` version
```
$ btcli --version
```

#### check `configuration`
```
$ btcli config --help
```
(~/.bittensor/config.yml)

$ btcli wallet create
